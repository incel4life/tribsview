# tribsview
a video viewer built for **tributes**

## features
* **shuffle**: randomize videos
* **infinite individual looping**: each video loops independently
* **queue**: control what plays next
* **zoom**: enlarge smaller videos
* **collections**: organize videos into tiles
* **auto-save**: changes save automatically
* **yt-dlp integration**: paste tiktok / instagram / x links to fetch media
* **single file export**: share collections as a single file

## what tribsview is not
* a video editor
